démarrer l'ensemble des conteneurs en arrière-plan

docker-compose up -d

voir le statut de l'ensemble de votre stack 

docker-compose ps

afficher les logs de votre stack

docker-compose logs -f --tail 5 

mettre à jour les images ou une image spécifique en ajoutant son nom à la fin de la commande

docker-compose pull

arrêter l'ensemble des services d'une stack

docker-compose stop

détruire l'ensemble des ressources d'une stack 

docker-compose down

valider la syntaxe de votre fichier docker-compose.yml

docker-compose config

supprimer les image non utilisées avec l'option -a 

docker image prune -a
