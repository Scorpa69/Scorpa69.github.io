REPOSITORIES

nano /etc/apt/sources.list et vérifier qu'il y a bien ce qui suit :

deb http://deb.debian.org/debian bullseye main contrib non-free

deb-src http://deb.debian.org/debian bullseye main contrib non-free

deb http://deb.debian.org/debian-security/ bullseye-security main contrib non-free

deb-src http://deb.debian.org/debian-security/ bullseye-security main contrib non-free

deb http://deb.debian.org/debian bullseye-updates main contrib non-free

deb-src http://deb.debian.org/debian bullseye-updates main contrib non-free

SIMULER MAJ OS COMPLETE :

apt full-upgrade -s

Paquet ont étés conservé : pour installer quand même faire

apt dist-upgrade

PROBLEME CLE GPG ABSENTE:

apt install gnupg

gpg --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys 467B942D3A79BD29

gpg --export --armor 467B942D3A79BD29 | apt-key add -

LISTER PAQUET INSTALLé:

dpkg -l

CHERCHER PROGRAMME INSTALLé

dpkg -l | grep selinux ou dpkg -l | grep grafana

SUPPRIMER LE LIEN REPOS AJOUTÉ POUR UNE APPLI : 

A faire après le "apt remove"

dpkg --purge zabbix-release ou dpkg --purge grafana-enterprise

INSTALLATION APPLICATION:

apt install nom_paquet

REINSTALLER UN PAQUET:

apt reinstall nom_paquet

SUPPRIMER PAQUET SIMPLE:

apt remove nom_paquet

SUPPRIMER PAQUET ET DEPENDENCES:

apt-get remove --auto-remove grafana-enterprise

SUPPRIMER PAQUET COMPLET: incluant les fichiers de configuration

apt purge nom_paquet

SUPPRESSION  ARCHIVES PAQUET EN GARDANT LA VERSION ACTUEL:

apt autoclean

SUPPRESSION ARCHIVES TOUS LES PAQUET:

apt clean

 

RECHERCHE D'UN PAQUET:

apt search nom_paquet

INFOS PAQUET:

apt show nom_paquet

RAISONS PAQUET QUI NE VEUT PAS S'INSTALLER:

apt why-not nom_paquet
