VOIR LE CONTROLLER:

lspci | grep RAID

 

AJOUTER LE DEPOT LSI:

nano /etc/apt/sources.list puis ajouter la ligne : deb http://hwraid.le-vert.net/debian bullseye main

 

SIGNATURE PAQUETS:

wget -O - https://hwraid.le-vert.net/debian/hwraid.le-vert.net.gpg.key | apt-key add -

 

INSTALL COMPOSANT:

apt update

apt install megacli

apt install megaclisas-status

 

Get all adapters status and config:

megacli -AdpAllInfo -aAll

 

Logical drive 0 on adapter 0 status and type

megacli -LDInfo -L0 -a0

 

Show physical disks from first controller:

megacli -PDList -a0

 

STATUT AVEC MEGACLISAS-STATUS:

megaclisas-status

 

STATUT EN 1 LIGNE :

megaclisas-status --nagios
