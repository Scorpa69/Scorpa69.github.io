Méthode 1 :

apt install -y unattended-upgrades
dpkg-reconfigure --priority=low unattended-upgrades

et sélectionner YES

Méthode 2 :

apt install -y unattended-upgrades apt-listchanges

nano /etc/apt/apt.conf.d/50unattended-upgrades décommenter les ligne suivantes :

"origin=Debian,codename=${distro_codename},label=Debian-Security";

Unattended-Upgrade::Automatic-Reboot "false"; => et passer à true

Unattended-Upgrade::Automatic-Reboot-Time "07:00"; => choisir l'heure du reboot

unattended-upgrades -v => pour tester le fonctionnement

nano /etc/apt/apt.conf.d/20auto-upgrades puis ajouter les deux ligne suivantes:

APT::Periodic::Update-Package-Lists "1";

APT::Periodic::Unattended-Upgrade "1";

systemctl enable unattended-upgrades

POUR VOIR LE LOG:  

cat /var/log/unattended-upgrades/unattended-upgrades.log
