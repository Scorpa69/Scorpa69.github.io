UFW:

Activer le pare feu

sudo ufw enable

Connaître le profil defaut du pare feu

sudo ufw status verbose

Liste des port ouvert

ufw status numbered

Définir le profil défaut

sudo  ufw default deny incoming

sudo ufw default allow outgoing

Ouvrir des ports

sudo ufw allow 10050/tcp

sudo ufw allow 443/tcp

sudo ufw allow 22/tcp

ufw allow 80/tcp

ufw allow 60000:61000/tcp

ufw allow ssh

ufw allow ftp

Fermer des ports

sudo ufw delete allow ssh

sudo ufw delete allow 443/tcp

Ouvrir pour un hôte ou un réseau spécifique et port spécifique

ufw allow from 203.0.113.0/24 proto tcp to any port 22

ufw allow from 203.0.113.103 to any port 873

Liste des appli reconnu par ufw contenant Nginx par exemple

ufw app list | grep Nginx
