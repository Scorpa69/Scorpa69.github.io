chmod -R xxx /dossier(/fichier)

    0 : (000) Aucune permission.

    1 : (001) Autorisation d’exécution.

    2 : (010) Droit d’écriture.

    3 : (011) Droits d’écriture et d’exécution.

    4 : (100) Droit de lecture.

    5 : (101) Droits de lecture et d’exécution.

    6 : (110) Droits de lecture et d’écriture.

    7 : (111) Droits de lecture, d’écriture et d’exécution.

 

chown -R user:user /dossier(/fichier)

-R pour avoir le récursif (dossier sous dossier et fichier modifier)

chmod [autorisations] [fichier / répertoire] [options]

             Symbole                     Description

Identités        u        C'est le propriétaire du fichier ou le répertoire. 

g        C'est le groupe du fichier ou du répertoire. 

o        D'autres utilisateurs, le reste du monde. 

a        Tout le monde - propriétaire, groupe et autres.

Permis                r        Accès de lecture.

w        Accès en écriture.

x        Accès d'exécution.

Actions                +        Ajouter permis.

-        Élimine les permis.

=        Le seul permis.

 

Commande                             Description                         Résultats

chmod u+rwx photo1.png                Ajouter toutes les autorisations pour le propriétaire.                rwx------

chmod g+x photo1.png                Ajouter la permission d'exécution pour le groupe.              rwx-x---

chmod o+r photo1.png                Ajouter la permission de lecture pour les autres utilisateurs.             rwx-xr--

chmod u-rw photo1.png                Supprimer les permissions de lecture et écriture pour le propriétaire.              --x-xr---

chmod a=r photo1.png                Nous établissons lecture comme seule  autorisation pour les 3 groupes.              r--r--r--

chmod u+rx,o+x photo1.png        Nous ajoutons les autorisations lecture et exécution pour le propriétaire et exécution pour les autres.           ---------
