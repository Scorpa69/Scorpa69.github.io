Augmenter la taille d'un disque d'un VM azure

  1. Si c'est le disque OS, éteindre la vm par le portail azure (pas par le vm sinon le disk os ne se libère pas pour l'agrandissement)
  2. Sur la page de la vm sélectionner disk ou disques
  3. Sur la page des disques de la vm, sélectionner le disque à agrandir 
  4. Sur la page du disque sélectionner aller dans le menu "size and performance"
  5. Changer la taille du disque et/ou ou augmenter celui de l'existant s'il reste de la place pour le déployer
  6. Retourner sur la page de la vm puis démarrer ou start
  7. FIN
