CLI :

Se connecter en Telnet ou SSH

 

Config t

Int vlan #

TO REMOVE IP HELPER: no ip helper-address 10.92.50.1

TO ADD IP HELPER: ip helper-address 192.168.0.2

 

SUPPRIMER UN IP HELPER :

  1. connexion au switch
  2. show vlan
  3. show ip int vlan xx  => le faire pour chaque VLAN (10, 20, 40, 50, 60, 70, 90, 150, 160)
  4. enable
  5. conf t
  6. int vlan xx
  7. no ip helper-address 10.92.50.1
  8. exit
  9. refaire les étapes 5, 6, 7 sur chaue VLAN nécessaire
