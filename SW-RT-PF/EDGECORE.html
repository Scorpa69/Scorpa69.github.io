Pour sauvegarder les modif en WebUI :

  • Aller dans le menu Home\System\File Management\Copy Operation
  • Sélectionner "running-config to startup-config" à la place de "file to file"
  • Sélectionner le fichier de startup-config (pour savoir lequel c'est, il faut aller dans le menu "Set Start-Up" puis regarder le fichier noté YES pour le startup

 

Pous sauvegarder la config avec tftp :

  • Aller dans le menu Home\System\File Management\Copy Operation
  • file to tftp
  • Renseigner l'adresse du serveur tftp
  • Selectionner config et nommer le fichier comme désiré
  • Refaire pareil pour opcode

 

Pour rebooter le switch en CLI seulement :

  • Se connecter en telnet (admin admin)
  • Taper: reload
  • Confirmer
  • Attendre reboot
