REPARATION INSTANCE

  1. Vérifier la version de MS SQL en PROD en allant dans les services puis propriétés et regarder le chemin pour connaitre la version MS SQL et la version 32 ou 64 (MSSQL13 =>SQL2016)
  2. Charger le cd d'installation de la version SQL déterminer ci-dessus dans la VM
  3. Lancer le centre d'installation SQL dans la version souhaité
  4. Sélectionner Maintenance
  5. Sélectionner Réparer (il est possible qu'on demande le chemin du CD)
  6.  Sélectionner l'instance à réparer puis cliquer sur réparer
  7. Attendre
  8. Tester si le problème est toujours présent
