En root ds mysql: 

Passer en low:

SET GLOBAL validate_password.policy=LOW;

 

Passer en medium:

SET GLOBAL validate_password.policy=MEDIUM;

 

Voir le status:

SHOW VARIABLES LIKE 'validate_password%';

 

Passer à 7 caractere mini:

SET GLOBAL validate_password.length=7;

 

DANS LE FICHIER CONFIG EN MODE PERMANENT AJOUTER :

Pour zabbix : nano /etc/mysql/mysql.conf.d/mysqld.cnf

 

[mysqld]

 

validate_password.policy = MEDIUM

validate_password.length = 7
