Serveur PXE avec pfSense en tant que serveur DHCP :

Tout d'abord, vous devez installer pfSense sur un système dédié. Assurez-vous que le système dispose de deux cartes réseau, une pour la connexion Internet et une pour la connexion au réseau local.

Connectez-vous à l'interface Web de pfSense en utilisant un navigateur Web. L'interface Web est accessible à l'adresse IP de la carte réseau LAN de pfSense.

Configurez les paramètres réseau de pfSense en utilisant l'assistant de configuration. Configurez l'interface LAN avec une adresse IP statique et activez DHCP sur l'interface. Pour ce faire, accédez à l'onglet "Interfaces" dans le menu principal, puis cliquez sur l'icône "Assigner des interfaces" pour configurer l'interface LAN.

Activez le serveur DHCP sur l'interface LAN. Pour ce faire, accédez à l'onglet "Services" dans le menu principal, puis cliquez sur l'icône "DHCP Server". Configurez les options DHCP pour votre réseau local, notamment l'adresse IP de la passerelle par défaut et des serveurs DNS.

Installez le serveur TFTP sur le système pfSense en utilisant le package "tftp-hpa". Pour ce faire, accédez à l'onglet "System" dans le menu principal, puis cliquez sur l'icône "Packages". Recherchez et installez le package "tftp-hpa".

Configurez le serveur TFTP en utilisant le fichier de configuration "/etc/default/tftpd-hpa". Définissez les options de configuration, notamment le répertoire racine pour les fichiers de démarrage PXE.

Téléchargez et configurez les fichiers de démarrage PXE pour les clients. Copiez les fichiers "pxelinux.0", "menu.c32", "vesamenu.c32" et "memdisk" dans le répertoire racine TFTP. Créez un fichier de configuration de menu PXE pour définir les options de démarrage pour les clients.

Activer le service NFS pour servir les fichiers aux clients. Pour ce faire, accédez à l'onglet "Services" dans le menu principal, puis cliquez sur l'icône "NFS Server". Configurez les options NFS pour partager le répertoire racine TFTP avec les clients.

Testez le serveur PXE en démarrant un client en mode PXE. Assurez-vous que le client est configuré pour démarrer en mode PXE et qu'il est connecté au réseau local. Lorsque le client démarre, il devrait recevoir une adresse IP et télécharger le fichier de démarrage PXE du serveur TFTP. Le client devrait ensuite être en mesure de démarrer l'installation du système d'exploitation à partir du serveur NFS.

Ces étapes devraient vous permettre de mettre en place un serveur PXE avec pfSense en tant que serveur DHCP. Cependant, la configuration exacte peut varier en fonction de votre environnement spécifique.
