Convertion VHDX vers VHD :

Convert-VHD -Path "C:\Users\A09\Desktop\W10_Professional_x64_fr-FR_MSDN.vhdx" -DestinationPath "d:\DD_VM\W10_Professional_x64_fr-FR_MSDN.vhd"

Pour changer de type de disque on peut ajouter l'option "-VHDType" suivi du type souhaité : Fixed, Dynamic ou differencing

 

Convertion VHD vers VHDX :

Convert-VHD -Path "e:\folder\disk.vhd" -DestinationPath "e:\folder\disk.vhdx"

Pour changer de type de disque on peut ajouter l'option "-VHDType" suivi du type souhaité : Fixed, Dynamic ou differencing
