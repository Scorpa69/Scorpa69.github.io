Liste DHCP dans un domaine :

netsh dhcp show server

Suppression d'un DHCP dans le domaine :

netsh dhcp delete server NomCompletDuServeur AdresseIPduServeur

MIGRATION DHCP en PS Admin :

  1. sur le DHCP destination : installer le rôle DHCP (ne pas terminer la config tout se suite)
  2. sur le DHCP source : Export-DhcpServer -ComputerName VIDFADS01 -File "C:\Users\a09_admin\desktop\DHCP-Config.xml" -Force
  3. sur le DHCP source : Stop-service -name "DHCPServer"
  4. sur le DHCP source : Set-Service -Name DHCPServer -StartupType "Disabled"
  5. sur le DHCP source : Copy-Item -path "C:\Users\a09_admin\desktop\DHCP-Config.xml" -destination "\\vidfads02\c$\Users\a09_admin\desktop\DHCP-Config.xml"
  6. sur le DHCP destination : Get-Service -Name "DHCPServer" si arrêté le mettre en route Start-service -name "DHCPServer"
  7. sur le DHCP destination : Terminer la config du DHCP dans la console serveur ou depuis la console DHCP, autoriser le serveur DHCP dans l'AD en cliquant droit sur le serveur
  8. sur le DHCP destination : Import-DhcpServer -ComputerName VIDFADS02 -File "C:\Users\a09_admin\desktop\DHCP-Config.xml" -Force
  9. sur le DHCP destination : Restart-service -name "DHCPServer"
  10. Un fois le fonctionnement validé sur la destination (attendre 48h)
  11. Désinstaller le rôle DHCP sur le serveur source

En cas de retour arrière : Set-Service -Name DHCPServer -StartupType "Automatic" puis Start-service -name "DHCPServer" pour remettre en route le source
