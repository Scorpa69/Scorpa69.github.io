Lancer cmd avec la commande run as 

PROBLEME APPLICATION : pour repérer le cache de l'appli et le vider

dir *nom du fichier* /s

dir *firefox* /s

dir *autodesk* /s

dir *outlook* /s

dir *azure* /s

 

SUITE MAJ OU ARRET BRUSQUE DE LA MACHINE :

sfc /scannow

Si erreur non réparé : Dism /Online /Cleanup-Image /RestoreHealth

 

PROBLEME DISQUE: (A faire une fois par mois)

chkdsk C: /f

 

Problème de droit user ou autre : A faire quand problème sur machine user

gpupdate /force

redemarrage machine

 

Pour lancer les outil diag microsoft : voir https://docs.microsoft.com/fr-fr/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/ee424379(v=ws.11)#available-troubleshooting-packs

msdt.exe /id PerformanceDiagnostic

 

Voir l'état de  l'image Windows :

 DISM.exe /Online /Cleanup-image /Scanhealth

 

Réparer l'image Windows :

DISM.exe /Online /Cleanup-image /Restorehealth

 

Réparation démarrage WS2008r2 (ou Win7) :

Faire un chkdsk c: /f

Utiliser la commande bootrec /set nointegritycheck

 

Réparation WMI:

Etat WMI: winmgmt /verifyrepository

Correction des erreurs: winmgmt /salvagerepository

Repository: winmgmt /resetrepository (ce qui implique qu’il faudra réimporter les classes que les différentes applications ont pu installés sur la machine)
