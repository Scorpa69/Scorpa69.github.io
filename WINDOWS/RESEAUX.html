PORT OUVERT :

Connaitre les ports ouvert sur une machine :

netstat -ano ou -abno

 

Connaitre les ports en listening :

netstat -aon | findstr /i listening

netstat -ano | find ":443"

ROUTAGE

Ajout Route vers le reseau local distant:

Add-VpnConnectionRoute -ConnectionName "PERSO2" -DestinationPrefix 192.168.33.0/24 -RouteMetric 200 -PassThru 

 

Suppression Route vers le reseau local distant:

Remove-VpnConnectionRoute -ConnectionName "PERSO" -DestinationPrefix 192.168.33.0/24 -PassThru

 

AFFICHIER LISTE ROUTE :

route print

 

AJOUTER UNE ROUTE NON PERSISTENTE :

route add 192.168.230.0 mask 255.255.255.0 192.168.1.250

 

AJOUTER LA GATEWAY 13.31.50.254 EN ROUTE PAR DEFAUT ET PERSISTENTE:

route /p add 0.0.0.0 mask 0.0.0.0 10.31.51.254 

 

SUPPRIMER LA GATEWAY :

route delete 0.0.0.0

 

CHANGER GATEWAY :

route change 0.0.0.0 mask 0.0.0.0 10.31.51.254

 

CHANGER METRIQUE:

route add 192.168.230.0 mask 255.255.255.0 192.168.1.250 METRIC 100

BROADCOM TEAMING (WS2008)

Configurez le regroupement

 

Exécutez Broadcom Advanced Control Suite 4.

Sélectionnez TEAM VIEW dans le menu Filtre de la barre de menus.
 

Cliquez avec le bouton droit de la souris sur Équipes dans l’arborescence de gauche et Créer une équipe.
 

Lorsque l’Assistant Regroupement Broadcom s’affiche, cliquez sur Suivant.

Sur l’écran Nom de l’équipe, saisissez n’importe quel nom d’équipe, puis cliquez sur Suivant.
 

Sur l’écran Type d’équipe, sélectionnez n’importe quel type d’équipe, puis cliquez sur Suivant.
 

Sur l’écran Assigner des collaborateurs, sélectionnez les cartes NIC que vous souhaitez ajouter au groupe, puis cliquez sur Ajouter. Une fois que toutes les cartes NIC sont sélectionnées, cliquez sur Suivant.
 

Sur l’écran Désigner un membre de secours, sélectionnez Use the following member... si vous souhaitez définir une carte réseau de secours à partir de l’équipe, puis cliquez sur Suivant.
Si vous souhaitez définir clairement des cartes NIC actives et en veille, sélectionnez Enable Auto-Fallback....
 

Sur l’écran Configuration de LiveLink, configurez le LiveLink si nécessaire, puis cliquez sur Suivant.

Sur l’écran Configure VLAN Support, configurez le VLAN si nécessaire, puis cliquez sur Suivant.

Lorsque le message « Félicitations ! » s’affiche, sélectionnez Commit changes to... pour enregistrer les paramètres, puis cliquez sur Terminé pour fermer l’Assistant.
Si vous continuez à configurer un autre regroupement, sélectionnez Enregistrer les modifications et....

La configuration du regroupement peut prendre quelques minutes en fonction de l’environnement.
Notez également que la connexion réseau sera temporairement déconnectée lors de la configuration du regroupement.

Une fois la configuration terminée, vérifiez que l’option « BASP Virtual Adapter » s’affiche sur les connexions réseau.
