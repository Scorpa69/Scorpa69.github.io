Get-ADComputer -Filter * -properties * | where {$_.OperatingSystem -ilike “Windows Server 2012 *”} | Select Name, OperatingSystem

 

 

TOUT LES SERVEUR AVEC MSSQL :

Get-ADComputer -filter 'serviceprincipalname -like "*mssql*" -and operatingsystem -like "Windows Server*"' -properties ipv4address | ft name,ipv4address

 

 

LISTE DES CONNEXIONS SANS DHCP AVEC DNS CONFIGURE D'UN PC:

Get-WmiObject Win32_NetworkAdapterConfiguration | Where-Object {$_.DHCPEnabled -ne 'True' -and $_.DNSServerSearchOrder -ne $null} | Export-Csv c:\users\A09\desktop\IPFxeAC.csv

 

 

LISTE MACHINE WINDOWS SERVER DANS CSV :

Get-ADComputer -Filter '(OperatingSystem -like "Windows Server*")' -properties * | Sort-Object Name | select-object Name, Operatingsystem, Enabled, IPV4Address| Export-Csv c:\users\A09\desktop\ListServeur.csv

 

POUR TROUVER SI DES LOGICIEL CONTENANT DELL DS LE NOM SONT INSTALLE:

Get-WmiObject -Class Win32_Product | where-object name -match "^dell.*"

 

POUR LISTER TOUS LES PROGRAMME INSTALLE:

Get-WmiObject -Class Win32_Product
