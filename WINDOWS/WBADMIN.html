Etat live de la sauvegarde (ctrl c pour sortir)

Wbadmin get status

 

Liste des sauvegarde dans un lecteur:

wbadmin get versions -backupTarget:Z:

 

Suppresion sauvegarde version ou x version plus récentes :

Wbadmin delete backup -keepVersion:11/10/2015-06:48 -BackupTarget:F：machine:WIN-9814GD4FH95

Wbadmin delete backup -Version:11/10/2015-06:48 -BackupTarget:Z:

wbadmin delete backup -keepVersions:12 -BackupTarget:Z:

 

Arreter une sauvegarde en cours:

Wbadmin stop backup

wbadmin stop job

 

Lancer une sauvegarde avec les paramètre de sauvegarde planifiées : (sans paramètre)

Wbadmin start backup
