Installation SNMP sur W10 :

Info sur l'installation : Get-Service -Name snmp*

Avant 1803 : Enable-WindowsOptionalFeature -online -FeatureName SNMP

Après 1803 : Add-WindowsCapability -Online -Name "SNMP.Client~~~~0.0.1.0"

et pour l'enlever :Remove-WindowsCapability -Online -Name "SNMP.Client~~~~0.0.1.0"

 

Installation SNMP sur serveur Core : depuis CMD il faut taper powershell pour passer en Powershell

Get-WindowsFeature *SNMP* ==> Savoir si le service est déjà installé

Install-WindowsFeature SNMP-Service -IncludeAllSubFeature -Verbose ==> Commande pour l'installer
