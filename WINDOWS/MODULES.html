Pour trouver les modules qui sont installés dans un emplacement par défaut, mais qui ne sont pas encore dans votre session :

Get-Module -ListAvailable

Pour rechercher les modules qui ont déjà été importés dans votre session, à l’invite PowerShell :

Get-Module

Commande d'un module :

Get-Command -Module <module-name>

Importer un module :

Import-Module <module-name>

Installer un module :

install-Module -Name <module-name>
